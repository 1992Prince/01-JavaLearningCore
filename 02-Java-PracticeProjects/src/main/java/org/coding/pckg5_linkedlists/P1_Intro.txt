ğŸ“’ Notes: Linked List vs Array

1ï¸âƒ£ Nature

Array â†’ Linear DS, elements contiguous memory me store hote hain.
Linked List (LL) â†’ Linear DS, elements random memory me store hote hain but connected via pointers (references).

2ï¸âƒ£ Memory

Array â†’ Contiguous memory, memory efficient (sirf values).
Linked List â†’ Non-contiguous memory, har node me extra space lagta hai (pointer ke liye).

ğŸ‘‰ Node structure in LL: [data | next address]
ğŸ‘‰ Array element: [value]


3ï¸âƒ£ Size

Array â†’ Fixed size (compile-time decide hota hai, ya resize karna padta hai).
Linked List â†’ Dynamic size (jitni zarurat ho utne nodes add/delete kar sakte ho).

4ï¸âƒ£ Fetching Elements

Array â†’ Fast (direct index se access).
Linked List â†’ Slow (sequential traversal karna padta hai, ek ek karke).

5ï¸âƒ£ Insertion & Deletion

Array â†’ Difficult (shift karna padta hai elements ko, time lagta hai).
Linked List â†’ Easy (sirf pointers change karne hote hain).

6ï¸âƒ£ Time Complexity Table

| Operation         | Array                   | Linked List            |
| ----------------- | ----------------------- | ---------------------- |
| Access (by index) | **O(1)** (direct index) | **O(n)** (traverse)    |
| Search (by value) | **O(n)**                | **O(n)**               |
| Insertion at end  | O(1) / O(n)\*           | O(n) (or O(1)\*\*)     |
| Insertion at head | O(n) (shift needed)     | O(1)                   |
| Deletion at head  | O(n) (shift needed)     | O(1)                   |
| Deletion at end   | O(1) / O(n)\*           | O(n) (unless tail ptr) |

* Array me agar space full hai to resizing O(n).
** LL me agar tail pointer maintain karte ho to O(1), warna O(n).

7ï¸âƒ£ Why differences?

Insertion/Deletion

    - Array â†’ elements shift karne padte hain â†’ costly.
    - LL â†’ pointers update hote hain â†’ easy.

Access

    - Array â†’ direct address calculate hota hai: Base + index * size.
    - LL â†’ har node traverse karna padta hai â†’ slow.

Memory

    - Array â†’ only data store.
    - LL â†’ data + extra pointer store (extra memory overhead).
